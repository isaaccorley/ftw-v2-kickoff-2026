# Fields of The World Tech Kickoff

Slidev deck for a 15-minute Fields of The World update.

## Run

```bash
make install
make serve
```

Local server: `http://localhost:3030`

## Build

```bash
make build
```

For GitHub Pages:

```bash
bun run build --base ./ --router-mode hash
```

## Files

- `slides.md` - main deck
- `style.css` - TG/FTW theme styling
- `public/` - fonts and imagery copied from the pangeo scaffold plus FTW public assets
